# FastAPI Task Manager

A Task Manager API built using FastAPI.

## Features

- User Registration
- JWT Authentication
- Create Tasks
- Get Tasks
- Update Tasks
- Delete Tasks
- Pagination
- Filtering

## Run Locally

1. Clone the repository

git clone https://github.com/Yuvika-A/fastapi-task-manager.git

2. Navigate to project folder

cd fastapi-task-manager

3. Install dependencies

pip install -r requirements.txt

4. Run the server

uvicorn app.main:app --reload

5. Render (Deployment)
https://fastapi-task-manager-wqws.onrender.com/docs#/
